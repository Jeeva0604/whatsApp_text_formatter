## 1.1.0
-   A Flutter package for WhatsApp-style text formatting. Supports:
- **Bold** (`**text**`)
- *Italic* (`_text_`)
- ~~Strikethrough~~ (`~~text~~`)
- `Monospace` (`` `text` ``)
- Bulleted Lists (`* text` or `- text`)
- Numbered Lists (`1. text`)
- Blockquotes (`> text`)
- Inline Code (` `text` `)
